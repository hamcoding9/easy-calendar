# 🗓️Easy Calendar
## Topic
* 일정 제목에서 자동으로 시간을 가져와 네이버 캘린더에 등록해줍니다.
## Features
* 네이버 로그인 후 캘린더와 연동
* 일정 제목에 시간을 입력하면 자동으로 해당 시간에 일정 등록
